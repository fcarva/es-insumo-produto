## Fluxo do dia a dia
Dentro da pasta `es-insumo-produto/`:

```bash
git add -A
git commit -m "sua mensagem"
git push            # já está com upstream (origin/main)
```

## Conferir
```bash
git remote -v
git log --oneline -5
git ls-remote --heads origin
```

## Visibilidade
O repositório foi criado por você no GitHub. Confirme em
**Settings → General → Danger Zone** que a visibilidade está **Public**
(opção escolhida). Não consigo ler/alterar isso a partir daqui.

## (Opcional) Release citável da entrega da disciplina
```bash
git tag -a v1.0-disciplina -m "Versão entregue à disciplina, 2026/1"
git push origin v1.0-disciplina
```
No GitHub: Releases → Draft a new release → escolha a tag → anexe o PDF
compilado. Habilitando o Zenodo, o release ganha DOI citável.
